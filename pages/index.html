Ultimate Stopwatch

Watches

The Ultimate Stopwatch™ is a professional timing tool for the iPhone and iPad, providing multiple stopwatches for anyone who has to track a large number of events, especially coaches, trainers, or PE teachers. Designed for iOS7.

Simple to use, but powerful enough to handle a heavy workload.

Download today

No Limits

  • Time an unlimited number of watches (with up to 90 visible at a time on an iPad)
  • Track unlimited laps per watch (last 4 to 10 displayed, rest shown on reports)
  • Track unlimited number of races, each named with location and any notes
  • Have your Watches run all the time (independently of whether app is running or not); no power required when watches are not displayed on screen

Complete control

  • Edit built-in activity list to include whatever sports/events you track
  • Control each watch separately, or use master start/stop/reset
  • Choose from 17 different stopwatch formats (iPhone/iPad, bigger/smaller, more/less info shown)
  • Switch at any time between split/cumulative display
  • Control the fractions of a second to display (tenths, hundreds, thousandths)

Unlimited People

  • Organize your people into teams, classes, or groups
  • Report for each person on what their races/times have been across all races
  • Import people from your contacts, including photos and emails

Flexible Reports

  • Add notes to each race or even each person in a race
  • Sort reports by time or by name
  • Print reports or copy/paste into other applications
  • Send reports via email to each person or to the whole group
  • Track and report which type of activity was timed (even separately for each watch)
  • Find a race by name or use the map to find it

Easy to use

  • Load a race for a whole group in a single click
  • Auto-sort watches: as each person completes a lap, they drop to the bottom of the active list, thus it’s easier to find them in a large group
  • Or manually drag watches into whatever order you like
  • Track photos for each person to help identify during a large group run
  • Add, sort, or delete watches, even while race is ongoing
  • Correct any timing mistakes by easily changing any time for any watch or lap,
  • Undo any mistakes throughout the app (such as on a false start or hitting the wrong lap button: just shake and accept the undo)
